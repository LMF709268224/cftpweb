# CFTP Documentation

本目录是 CFTP 项目的统一文档中心，主要用于存放项目的架构设计、业务流程说明、开发规范以及部署指南等。
所有的核心参考资料及相关沉淀文档均已按照类别归档在此，以便于后续团队成员快速入手与排查问题。

## 目录结构

* **[架构设计](./架构设计/)**: 架构设计与规范
  * 涵盖了前后端分离架构、BFF 网关的设计思想。
  * 包含了 API 规范（如分页 `list-and-count-plan` 准则）。
  * 涵盖了微服务的对接更新记录。
* **[业务流程](./业务流程/)**: 业务流程说明
  * 考生端与管理端的核心操作流程梳理。
  * 各种能力流转图谱及历史测试验收的问题复盘。
* **[开发维护](./开发维护/)**: 开发与维护指南
  * 开发规范、前端组件（如红点 UI `red-dot-badge`）的需求说明。
  * `candbff`、`adminbff` 相关的 Code Review 及重构指南。
  * 接口兼容性（Fallback）审计记录等。
* **[运维部署](./运维部署/)**: 系统架构与运维部署指南
  * 包含各依赖中间件（Redis, PG, NATS 等）的初始化。
  * 第三方服务（Casdoor）的集成与配置等。
